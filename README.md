# MITx 6.86x — Machine Learning with Python: From Linear Models to Deep Learning

This repository documents my learning progress for **MITx 6.86x**, part of the MITx MicroMasters Program in Statistics and Data Science.

The course focuses on building a strong foundation in **machine learning algorithms**, from linear models to deep learning and reinforcement learning.

---

## 📚 Course Overview

- Institution: Massachusetts Institute of Technology (MITx)  
- Platform: edX  
- Status: In Progress  
- Focus: Machine learning, modeling, and algorithm implementation  

---

## 🧠 Topics Covered

- Linear models and regression  
- Classification (Perceptron, SVM)  
- Feature engineering (bag-of-words)  
- Optimization (gradient descent)  
- Neural networks and backpropagation  
- Unsupervised learning (clustering, GMM)  
- Collaborative filtering  
- Reinforcement learning (Q-learning, DQN)  

---

- Jupyter Notebook / Google Colab

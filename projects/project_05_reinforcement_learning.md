# Project 5 — Reinforcement Learning (Q-learning & Deep Q-Networks)

## 📌 Overview

This project focuses on learning control policies for a text-based game using reinforcement learning.

The agent interacts with a simulated "Home World" environment where states are described in text.  
Different Q-learning approaches are implemented to learn optimal actions based on rewards.

Achieved 100% accuracy on evaluation tasks.

---

## 🎯 Objectives

- Implement tabular Q-learning for simple environments
- Apply Q-learning with linear function approximation
- Build and train a Deep Q-Network (DQN)
- Use bag-of-words representation for text-based states
- Learn optimal policies from reward-based interactions

---

## 🧠 Key Concepts Applied

- Reinforcement Learning (RL)
- Q-learning algorithm

- Exploration vs exploitation
- Function approximation
- Deep Q-Networks (DQN)
- State representation using bag-of-words
- Policy learning in partially observable environments

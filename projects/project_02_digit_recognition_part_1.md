# Project 2 — Digit Recognition (MNIST Classification)

## 📌 Overview

This project focuses on classifying handwritten digits using the MNIST dataset.  
The task involves applying multiple machine learning models to image data and comparing their performance.

The dataset consists of grayscale images of handwritten digits (0–9), each represented as a 28×28 pixel grid.

---

## 🎯 Objectives

- Implement linear and non-linear classification models  
- Apply machine learning techniques to image data  
- Compare model performance across different approaches  
- Understand the role of feature transformations  
- Work with real-world high-dimensional datasets  

---

## 🧠 Key Concepts Applied

- Linear regression for classification  
- Support Vector Machines (SVM)  
- Multinomial (Softmax) regression  
- Feature transformations and engineering  
- Kernel methods:
  - Polynomial kernel  
  - Gaussian (RBF) kernel  
- Dimensionality reduction (PCA)  
- High-dimensional data representation (784 features per image)  
- Model evaluation and comparison  

---

## 🛠️ Work Performed

- Loaded and processed MNIST image dataset  
- Converted images into flattened feature vectors  
- Implemented multiple classification models  
- Applied kernel methods for non-linear decision boundaries  
- Used PCA for dimensionality reduction  
- Trained and evaluated models on digit classification tasks  
- Compared performance across different approaches  

---

## 📊 Skills Demonstrated

- Working with image-based datasets  
- Applying machine learning models in Python  
- Feature engineering and transformation  
- Dimensionality reduction (PCA)  
- Model evaluation and comparison  
- Handling high-dimensional data
